# Pet Meme Generator - Final Results

The **Universal Meme Factory** is fully operational with the latest requirements.

## Pipeline Features
1.  **Intermediate Cleanup**: Temporary "clean" images are generated for the transition step but automatically deleted after processing.
2.  **Output Naming**: Final memes are saved as `[pet]_meme_style_[#].jpg`.
3.  **Caption Placement**: Text is anchored to the bottom edge, preventing subject obstruction.
4.  **Auto-Sizing**: Text wraps and scales dynamically to fit image width.
